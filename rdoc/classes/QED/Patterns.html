Patter Advice (When)

This class tracks When advice defined by demo scripts and helpers. It is instantiated in Scope, so that the advice methods will have access to the same local binding and the demo scripts themselves.

    def call(match, *args)
      @when.each do |(pattern, proc)|
        case pattern
        when Regexp
          regex = pattern
        else
          regex = when_string_to_regexp(pattern)
        end
        if md = regex.match(match)
          proc.call(*md[1..-1])
        end
      end
    end
