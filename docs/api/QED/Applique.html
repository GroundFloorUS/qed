Applique

Applique is a module built per-script from the applique dirctory. Applique scripts are loaded at the start of a session.

The Applique is whole collection of applique that apply to given demonstrandum. The applique that apply are the scripts located in the directory relative to the demonstrandum script and all such directories above this upto and the project’s root directory.

All scripts in the Applique must be compatible/consistant. For two demos to have separate applique must be kept in separate directores.
